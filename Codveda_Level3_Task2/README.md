# Customer Churn Analysis Dashboard

## Project Overview

This project presents an interactive **Customer Churn Analysis Dashboard** developed in **Power BI**. The dashboard analyzes customer behavior, identifies churn trends, and highlights the key factors influencing customer retention. It enables users to monitor important KPIs and make data-driven business decisions.

## Objectives

* Analyze customer churn patterns.
* Monitor key customer metrics.
* Identify factors contributing to customer churn.
* Support business decisions with interactive visualizations.

## Tools & Technologies

* Microsoft Power BI
* Microsoft Excel
* Data Cleaning & Transformation
* Data Visualization
* DAX (Data Analysis Expressions)

## Dashboard Features

* KPI Cards

  * Total Customers
  * Churn Customers
  * Churn Rate
  * Active Customers
* Customer Churn Distribution
* Churn by State
* Churn by International Plan
* Churn by Voice Mail Plan
* Customer Service Call Analysis
* Day, Evening, Night, and International Usage Analysis
* Interactive Slicers and Filters

## Key Insights

* Identified customer segments with the highest churn.
* Compared churn rates across different customer plans.
* Analyzed the impact of customer service calls on churn.
* Evaluated customer usage patterns to identify churn behavior.
* Enabled interactive filtering for detailed analysis.

## Dataset

The dataset contains customer information such as:

* State
* Account Length
* Area Code
* International Plan
* Voice Mail Plan
* Customer Service Calls
* Call Minutes
* Call Charges
* Churn Status
